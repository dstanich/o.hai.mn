O Hai Minnesota

Ahoy there.

The following is a currated list of Minnesota's online communities in the areas of tech, design, etc. It is maintained by @jimthedev in a Github repository. Please open an issue to add, modify or delete communities. If you run your own Slack community, we highly recommend using Slackin' to manage public invites. Enjoy! :)
